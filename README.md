# Interaktivni Sistem za Upravljanje Zaposlenima i Projektima (C++ / SQLite)

## Opis Projekta
Ovo je **backend desktop aplikacija u C++** koja omogućava interaktivno upravljanje zaposlenima i projektima u firmi. Projekat koristi **SQLite bazu podataka** za čuvanje informacija o zaposlenima, menadžerima i projektima.

Glavne funkcionalnosti uključuju:
- Dodavanje i uklanjanje radnika i menadžera
- Dodavanje i uklanjanje projekata
- Povezivanje zaposlenih sa projektima
- Pretragu zaposlenih po imenu, ID-ju ili poziciji
- Prikaz prosečne plate zaposlenih, radnika i menadžera
- Ažuriranje plata zaposlenih

## Tehnologije
Tehnologije
- C++17 / C++20 – korišćenje savremenih C++ standarda, uključujući moderne OOP paradigme, pametne pokazivače i lambda funkcije.
Objektno-orijentisano programiranje (OOP) – čista arhitektura sa klasama Zaposleni, Radnik, Menadzer i Projekat koja olakšava održavanje i proširenje koda.
- STL biblioteke – korišćenje Standard Template Library za efikasno upravljanje kolekcijama podataka, stringovima i algoritmima.
- SQLite3 – lagana, brza i pouzdana SQL baza podataka za backend skladištenje i manipulaciju podacima.
- Backend razvoj – fokusiran na stabilnu logiku i interakciju sa bazom podataka, spremno za eventualnu nadogradnju GUI-ja ili web servisa.
- Visual Studio / C++ IDE – razvoj i debugging koristeći profesionalne alate za C++.
